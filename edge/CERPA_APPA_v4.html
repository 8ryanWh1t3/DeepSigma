Command UI
⬡
Executive / Commander
See command health fast. Decide where attention goes.
Views Closure Integrity Score (CIS), Command Closure Confidence Index (CCCI), Patch Adhesion Rate (PAR), Deficiency Echo Score (DES). Approves, escalates, requests review. Does not edit operational evidence.
Dashboard · Review Summary · Trends · Brief Export
Ops UI
◈
Program Manager / Owner
Move findings from assignment to durable closure.
Drives findings through the CERPA cycle. Creates patches, uploads evidence, tracks Observation-to-Ownership Lag (OOL) and Decision-to-Evidence Gap (DEG). Requests closure but cannot self-certify.
My Queue · Finding Detail · Patch Builder · Evidence Panel
Command + Ops
◎
Reviewer / Board Member
Judge whether closure is real.
Evaluates Board Readiness Quotient (BRQ), Closure Gravity (CG), False Closure Rate (FCR). Approves, rejects, sends back. Scores readiness. Cannot rewrite raw observation history.
Review Board · Finding Detail · Evidence View · Recurrence History
Ops UI
◇
Analyst / Assessment Lead
Turn observations into structured truth.
Captures claims and events. Detects recurrence via Deficiency Echo Score (DES). Tracks Command Memory Retention (CMR). Prepares trend outputs. Cannot approve closure.
Intake Workspace · Findings List · Trend Explorer · Memory Lookup
Governance UI
⚙
Governance Admin / System Admin
Control roles, thresholds, workflow rules, and policy integrity.
Manages users, permissions, KPI threshold configuration, workflow locks, taxonomy. Views audit logs. Cannot alter historical closure records or rewrite canon.
Admin Console · Role Matrix · Threshold Config · Audit Logs
Governance UI
⊘
Auditor / Inspector / Oversight
Verify truth without changing canon.
Read-only access to findings, evidence chains, decision provenance. Exports audit packages. Traces closure integrity. Never modifies any operational record.
Audit Workspace · Evidence Chain · Export Center · Trace Viewer
