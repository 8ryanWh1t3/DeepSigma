The Problem

Every proposal role re-reads the same RFP independently. Notes drift. Amendments create scrambles. Things get missed.

The Solution

RFP Docs→ AI Extraction→ {} JSON→ Excel Tables→ Team Executes

One person parses the RFP with a structured AI prompt. Output is a single JSON file. Excel Power Query turns it into live tables. Each role pulls what they need. Amendments? Rerun and refresh.

Quick Start (3 min)

  1. Copy the extraction prompt into Co-Pilot
  2. Attach the RFP document
  3. Save the JSON output as rfp_extract.json
  4. Load into Excel via Power Query
  5. Share JSON file + notify the team

Why JSON?

  • Single source of truth — one file, one location
  • No redundant parsing — parse once, pull many
  • Reduced drift — everyone reads the same data
  • Auto-refresh — Ctrl+Alt+F5 updates all tables
  • Audit trail — every field has a page citation
