Exercise Walkthrough

A realistic lateral movement engagement mapped through CERPA

T+0h — CLAIM
Red team deploys lateral movement via compromised service account using Pass-the-Hash. Technique mapped to MITRE ATT&CK T1550.002. Exploitation evidence logged with timestamps, source host, and target credentials.
T+2h — EVENT
Blue SOC observes anomalous SMB authentication from unexpected subnet. Partial telemetry — endpoint agent missed initial credential access. Detection coverage gap documented.
T+4h — REVIEW
Joint red/blue adjudication session. Confirmed: exposure is real, detection coverage is partial. Mission impact assessed as HIGH. Board Readiness Quotient: 0.82. Contradiction flags raised on two prior risk acceptances.
T+8h — PATCH
Network segmentation rule deployed. Detection logic updated for credential relay patterns. Endpoint agent configuration hardened. Change control approval logged with authorizing official and evidence references.
T+24h — APPLY
Red retests — technique now blocked at network boundary. Evidence logged. Patch marked durable. Memory updated for future exercises. Regression baseline established.

What These Numbers Mean

Validated Exposures count adversary techniques that successfully executed against the environment. Each exposure has a full CERPA claim with ATT&CK mapping, execution evidence, and impact assessment.
Observed Events track blue team detections correlated to red claims. The ratio of events to exposures reveals detection coverage. A gap means adversary activity went unnoticed.
Review Closure Rate measures how many joint adjudication sessions produced a final decision. Open reviews indicate unresolved disagreements between red and blue assessments.
Regression Alerts fire when a previously patched vulnerability reappears in a subsequent exercise. Two alerts means two old exposures resurfaced — the patches did not hold.
