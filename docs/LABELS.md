# Coherence Ops Pilot Labels

This document defines labels used by the GitHub-native Coherence Ops pilot.

- `COH-DECISION`: Decision records and decision lifecycle work.
- `COH-ASSUMPTION`: Assumption tracking and refresh tasks.
- `COH-DRIFT`: Drift signal issues.
- `COH-PATCH`: Patch work that resolves drift.
- `COH-SEV1`: Critical drift with immediate risk.
- `COH-SEV2`: Significant drift requiring near-term patch.
- `COH-SEV3`: Low-severity drift for planned remediation.
- `COH-NEEDS-OWNER`: Item missing an explicit owner.
- `COH-NEEDS-SEAL`: Item missing or awaiting seal update.
- `COH-EXPIRED`: Assumption or decision dependency has expired.
