Three-Phase Demo

Baseline, drift injection, and patch — demonstrating the full coherence lifecycle.

Three clean episodes. No drift. Full memory coverage.

{

  "grade": "A",

Phase 2: Drift Injection

Outcome changes and confidence shifts trigger typed drift signals.

$ coherence demo --json | jq '.drift'
{
  "overall_score": 85.75,
  "grade": "B",
  "dimensions": [
    {"name": "policy_adherence",    "score": 100.0},

# Drift signals detected:

Phase 3: Patch

New episodes restore health. Score recovers toward baseline.
