Quickstart

Five steps to observable decisions.

1. Install

$ pip install deepsigma

2. Create a Session

from core import AgentSession

session = AgentSession("fraud-detector-v3")

3. Log Decisions

sealed = session.log_decision({
    "action": "quarantine_account",
    "reason": "Suspicious activity detected",
    "actor": {"type": "agent", "id": "fraud-detector-v3"},
    "targets": ["acc-12345"],
    "evidence": ["alert-789"],
    "confidence": 0.92,
})
# sealed episode has: episodeId, seal.sealHash, actor, actions, outcome

4. Score Coherence

report = session.score()
print(f"Score: {report['overall_score']}/100 ({report['grade']})")
# Score: 90.0/100 (A)

# Four dimensions measured:
#   policy_adherence    — are all episodes DTE-stamped?
#   outcome_health      — success and verification rates
#   drift_control       — drift signals vs episode count
#   memory_completeness — graph coverage of episodes

5. Detect Drift

signals = session.detect_drift({
    "action": "quarantine_account",
    "reason": "Different pattern detected",
    "actor": {"type": "agent", "id": "fraud-detector-v3"},
    "confidence": 0.45,  # dropped from 0.92
})
# returns: [{"driftType": "freshness", "severity": "red", ...}]

Framework Integration

REST Middleware

$ pip install deepsigma-middleware

from deepsigma_middleware import DeepSigmaMiddleware, configure

configure(agent_id="my-api")
app = DeepSigmaMiddleware(app)
# every HTTP request is now a sealed decision

Chain Runtime Bridge

$ pip install langchain-deepsigma

from langchain_deepsigma import ExhaustCallbackHandler

handler = ExhaustCallbackHandler(project="my-project")
chain.invoke(input, config={"callbacks": [handler]})
# every LLM call and tool use is logged

Agent Wrapper

$ pip install openai-deepsigma

from openai_deepsigma import DeepSigmaAgentWrapper

wrapper = DeepSigmaAgentWrapper(agent, session)
result = wrapper.run("What is the weather?")
# intent, tool calls, and completion are sealed
