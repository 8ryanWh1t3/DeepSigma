# Policy Packs (Scaffold)

Policy Packs are versioned, portable enforcement bundles for Σ OVERWATCH / RAL.

They allow you to ship governance as a **hashable artifact**:
- decisionType-specific DTE defaults
- degrade ladder rules
- blast-radius requirements
- verifier requirements
- tool allow/deny lists

## Files
- `policy_pack.schema.json` — JSON schema (scaffold)
- `packs/demo_policy_pack_v1.json` — example pack
